《叫魂》笔记

####第一章

  1. 叫魂：石匠们需要将活人的姓名写在纸片上，贴在木桩的顶部，这样会给大锤的撞击添加某种精神力量。
  2. 萧山事件—容易轻信的民众，受到贪婪的地方捕役中的恶棍以及无能的县府官员的挑动，而造成的又一桩司空见惯的丑事。
  3. 阻止公众为对付妖术而大动干戈——对比乌干达（英国）方法：惩罚为了威胁对手或为了制造让人畏惧的名声而＂假冒＂为巫师的人

政府没有支持巫术，而是同假冒巫术作斗争，然而普通民众没有能力辨别真假巫术，也没有被赋予这样的权力。于是政府就在保护巫术的旗帜下消灭它的敌人

####第二章

  1. 背景：1768年，康乾盛世。
    ①区域发展不平衡②人口流动，人口增长（玉米甜薯等作物的引进使得山地得以利用） ③扩大耕地面积导致的水土流失与生态灾难④商业发展➡阶层贫富分化➡流浪者（恐慌中被怀疑的对象）

  2. 对妖术的恐惧：那些无根无基的人们不受控制的流动而产生的焦虑

####第三章
皇帝：谋反与汉化——同一威胁的不同侧面（1755年胡中藻）剪辫的恐慌（薙发令）

满清帝国的问题：同时从普世主义角度和自身种族的角度出发表现其统治合法性➡处理谋逆的微妙处境(马朝柱谋反案：乾隆朝首次以复明为号召的运动，影响极大。)

文字狱：同可能造成群众性骚乱的大规模谋反事件相比，这一类尽在皇帝掌握中，可以大声颂扬满人的种族尊严而不必担心会引发什么意外。

官僚集团自身才是专制最大的障碍。

江南问题：行政机构效率下降

####第五章
妖术：通过对精神世界的操纵来强化个人力量。

  1. 认识上的强化力量（超越时间和空间，特别是预言未来）
  2. 遥控事物（穿越空间移动物体）
  3. 驾驭生死

整件事最令人厌恶的地方：术士以谋命为目的的作法程序竟然同挚爱孩子的父母所从事的礼仪活动有着相同的语言表述方式（对平常礼仪活动的嘲弄）

####第六章
各省的清剿—妖术与政治的关系

官僚责任制度的运作是围绕着对信息的控制而展开的

####第九章
中华帝国后期政治制度研究的两大传统：

  1. 着眼于行政官僚制的结构、人事和价值
  2. 注重于帝国的机制，特别是通讯制度的发展 ➡ 君主是帝国政治制度的一个部分而非权力无限的专制者

专制权力和常规权力之间的互动关系：

  1. 马克思韦伯—从长远看，后者取代前者（常规化和合理化）；二者相互冲突
  2. Hans Rosenberg—对普鲁士个案的研究，专制权力与常规权力不能长期共存
  3. ＂官僚君主制＂
  4. 克罗齐耶(Michel Crozier)—规则的扩展会限制他（皇帝）的权力，而太多的例外又会削弱他控制别人的权力像叫魂案这样一桩政治罪所造成的最大冲击，就在于它动摇了官僚们用以有效保护自己的常规行为方式，从而为弘历创造一个环境，使他得以就自己所关心的问题同官僚们直接摊牌。

政治罪将官僚生涯中的整洁有序的后院置于专制权力的严厉狂暴之下

官僚的抵制：①忙而不动②转移视线③统一步调（集体上奏）④常规化（卡尔•曼海姆Karl Mannheim——官僚思维的基本倾向是把所有的政治问题化约为行政问题）

####第十章

对这一事件的不同表达，取决于人们不同的社会角色及生活经历。从这一角度来看，叫魂主题被赋予不同变调，敷衍成不同的故事，每一个故事所表达的则是某一特定群体的恐惧。
——共同主题：伴随着未知人物和未知力量而来的凶险。

一旦官府认真发起对妖术的清剿，普通人就有了很好的机会来清算宿怨或谋取私利。

在这个权力对普通民众来说向来稀缺的社会里，以＂叫魂＂罪名来恶意中伤他人成了普通人的一种突然可得的权力。

（官僚制度）它的特性可以阻挡任何一种狂热。没有这样一个应急的锚碇，中国就会在风暴中急剧偏航。

附评：
这本书并没有黄仁宇的几本那么火，不知何故。文笔内容都很好，角度对我们来说也比较新颖。当然毕竟是洋人，角度不同也是很正常的事。如果不是专门研究的话，这样的书看一两本就差不多，再多就烦了。

《法学导论》读后感

《法学导论》可以说是我第一次与法学的零距离接触，故只好引申感想，泛泛而论。

97年大百科这一版在原书之外还增加了许多背景性质的材料。其中一篇为《拉德布鲁赫的生平及其思想历程》，对于我们理解全书内容和拉德布鲁赫其人有很大帮助。

在第一章中，作者对法律的概念和目的进行了总体上的表述，阐明习惯、法律和道德三者的关系，以及法哲学各流派和法律观念的讨论。除此以外，作者还提及法律语言和起源的问题，将历史法学派和唯物主义历史观点进行比较。通过第一章，作者将法律的全貌以一种非常概括的形式展现出来。而后的内容（除第十二章）则是依据不同的法律内容，同时结合当时德国的现实状况，进行分别论述。从当今中国的角度来看，很多内容难免显得有些过时。尽管如此，书中所揭示的观点仍然极具参考价值，特别是对于正处在法治社会建设当中的中国。

在宪法方面，作者通过从等级国家到专制国家再到立宪国家的变化，揭示出立宪国家的基础是自由，以及它的三个特征：民众参与立法、行政合法性、司法裁判独立。从这里我们可以看出，立宪国家并不是一个凭空产生的概念，而是建立在社会逐步发展的一个前后逻辑一致的基础上。对于欧洲大陆国家，专制国家是其形成自身国家精神的一个不可避免的过程，而英国则由于其历史的独特性，没有经过专制国家阶段就形成了其国家精神。（存疑）总而言之，在立宪国家阶段之前，独立国家精神是必不可少的。它为其后一个阶段中自由的充分发展和避免政党政治的分裂提供基础，凝聚社会共识，而这正是立宪国家中宪法所发挥的作用。对于当时的德国，“改革的根本困难在于它的不可分割的彼此相连的两个任务：德意志的新划分和统一国家的建设”。

在民商法方面，作者提到了私法的四个基础：私有财产权、契约自由、一夫一妻制和继承权。对德国而言，相较于日耳曼公法性质的财产权，私有产权概念是从罗马法中引入的，由此引发的“权利”概念构成整个法律体系的核心。此外在这一领域中还存在不断变化的经济活动现实与僵化法典之间的矛盾，甚至构成对法律安全的挑战。书中《德国民法典》的例子给我们一个很好的启示，“以多种多样的变化号召法官对法律进行有创造性的发展”。

上述民法主要针对的是个人主义法律时代，自由放任的个人主义经济时代。然而随着政府干预经济时代的到来，对社会法律的需求愈发强烈，而这新领域毫无疑问是属于经济法和劳动法的。虽然二者都立足于社会，但他们之间存在着较大的差异。首先是考察经济关系的切入点不同，前者“从国民经济生产率的角度”，而后者是“从保护经济弱者的角度”。其次是对象的不同，前者“倾向于保护企业主利益”，而后者则“倾向于保护劳动者利益”。在马克思主义的影响下，作者将更多的篇幅放在了劳动法上，并提出了新劳动法的构想：在人身自由的层次上实现劳动者的人权。

在刑法方面，作者从人道主义的立场出发，主张刑罚应偏重于教育。针对魏玛共和国的刑法改革，作者分刑罚体系、量刑、保安措施和刑事法官的地位四个方面论述了自己的理论。特别是在第一个方面中，作者认为在新刑法的体系中应当废除死刑，而这个观点在当今中国仍然可以说是超出主流意见之外的。他还主张废除名誉刑，因为这不仅加重社会的偏见和歧视，也给那些有前科的犯人增添重新融入社会的负担，进而促使他们再次犯罪。刑法的根本目的在于维持正常的社会秩序而不是刑罚本身，这一点对中国的刑法改革至关重要。

作者在书中专列一章讲国际法，不是仅仅从法律哲学的角度来论述，还谈及实现世界和平的基础。世界和平的障碍，一方面在于国家与国家之间缺乏共识，另一方面在于人们根深蒂固的对战争的信仰，而无论其持何种政治观点。而这两方面在21世纪的今天，仍然没有得到多少改善。虽然联合国的表现比国联要强很多，但是并没有达到最初的预想。

在最后一章“法学”中，作者对那些立志于从事法学领域工作的学生们揭示法学的任务，提供很多中肯的建议，而这让我们这些门外汉也能获得很多的启发。当然全书的内容远不止这些，但是对于我们这个至今不承认分权之合理性的国家来说，未免显得有些“难于理解”。“法治国家对我们来说不是一个政治概念，而是一个文化概念。”而在政治统领一切且法律意识淡薄的现实情形下，建设法治国家不能不说是任重而道远。

莫言

很久没写了，确实是懒惰啊。。。。

今天文学奖的结果出来了，是莫言的。我没读过莫言的书，也没听说过这个人。不过大家耳熟能详的几个作家我还是知道的，可见他在国内的影响力有多大，当然诺奖才不会考虑这个呢。所以，对于他获此奖是不是名至实归，我无权评判，也没有评判的兴趣。无论他是不是得奖，我都没有拜读其作品的打算，没那个闲情逸致。我说真的，大部分当代作家的小说我都没读过，兴趣不能说没有，只不过时间有限罢了。至于他的竞争对手村上，我倒是读过不少。故事虽然没一个完全看明白的，但是感觉很畅快。必须要承认的是，这种看法很肤浅，幸好再也不用做阅读了。还要补充一句，文学奖的热门人物居然是莫言和村上，呵呵。

不出意外，他的得奖很快引发了国内舆论的强烈关注，各路段子层出不穷。而更重要的是，他的东西肯定要大卖了，奖金和版税神马的，不知道国内其他各路作家会作何感想。竟然还有预计他要做中央委员，还有什么文化部部长，真是搞笑，诺奖在国内有那么大威力么，别YY了。

最后，谈谈诺奖吧。医学奖中又有一个日本人。远的说不清，今几年中自然科学方面的奖项总能看见日本人的身影。期间差距不言自明。倒不是说这些奖项中又个什么高低之分，只是觉得自然科学奖的评选相比较而言更加客观些罢了，而在这些需要硬功夫的领域里，我们要走的路还长着呢。至于这个文学奖，莎士比亚没拿，好像也没有妨碍他成为一代文豪。

明天的和平奖更令人期待，嘿嘿

真没想到这么久写一篇居然是谈这个的。
PS：没想到和平奖居然是欧盟。。。

二零一二年十月十五日

也许是因为下过雨的缘故，今日从图书馆出来的时候发觉人特别的少，再配合着些许朦胧的雾气，简直有种阴森的感觉。想来倒是适合某些人卿卿我我，可惜路灯不知何时忽然修好了，喷出明亮的光来，扑灭了这若隐若无的意境。

忽然两骑车少年从身旁飞驰而过，当然算不上白衣了。明亮的路灯可算是给他们一个极大的便利，于我这双目昏花的人亦如是。事物总有正反两面么毕竟……

回来的路上反正也无事（也可以说是无聊），便又想到了自己的专业。说句实在话，自己并没有用心读之，每每敷衍了事，草草收场而已。看到某前辈所转发的痛批本科教育之失的言论，顿时呵呵呵。这样的教育的确很不能叫人满意，然而若不如此，岂有我等阳奉阴违之人的活动空间？至于其他未来什么的，更是不必考虑了。当然，路并非没有，只是走起来的难度要比现在大上很多倍了。

补：搬家时把此文给翻出来了，再加上今天看的几篇论坛里的帖子，顿时感概万千。唯愿自己能不失本心
