测试hexo支持的markdown语法

区块元素

段落与换行

一个 Markdown 段落是由一个或多个连续的文本行组成，它的前后要有一个以上的空行（空行的定义是显示上看起来像是空的，便会被视为空行。比方说，若某一行只包含空格和制表符，则该行也会被视为空行）。

段内强制换行直接回车，不用加两个空格。（ GFM 特性）

标题

1到6个#，表示一到六级标题。#后要加一个空格

测试标题，“=”相当于h1

测试标题h2，“-”相当于h2

效果：

测试标题，“=”相当于h1

测试标题h2，“-”相当于h2

“=”和“-”也是有效的，不过要求其数量最少2个。“—”不会与分隔线混淆。
