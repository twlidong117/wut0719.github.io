测试hexo支持的markdown语法

区块元素

段落与换行

一个 Markdown 段落是由一个或多个连续的文本行组成，它的前后要有一个以上的空行（空行的定义是显示上看起来像是空的，便会被视为空行。比方说，若某一行只包含空格和制表符，则该行也会被视为空行）。

段内强制换行直接回车，不用加两个空格。

列表

无序列表用减号或星号都可以，但不要混用。有序列表用数字接一个英文句点。

  • 第一级
  • 第一级
    • 第二级
    • 第二级
    1. 第二级
    2. 第二级
      • 第三级
      • 第三级
      1. 第三级
      2. 第三级
  • 第一级
    列表的嵌套支持超过两级以上，缩进为两个空格。无序列表的标号会根据层级变化，有序列表的数字标号则不受影响。
